# react-native-traccar

package for traccar

## Installation

```sh
npm install react-native-traccar
```

## Usage

```js
import { 
    startTrackingService,
    stopTrackingService
 } from 'react-native-traccar';

// ...

startTrackingService()

// ...


stopTrackingService()

```
